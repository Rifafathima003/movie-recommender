# 🎬 Movie Recommendation System

## 📌 Overview
A content-based movie recommendation system using TMDB dataset.

## 🚀 Features
- Recommend top 5 similar movies
- Uses NLP (CountVectorizer)
- Cosine similarity

## 🛠 Tech Stack
- Python
- Pandas
- Scikit-learn
- Streamlit

## 📂 Dataset
TMDB 5000 Movies Dataset (Kaggle)

## ▶️ Run Locally
```bash
pip install -r requirements.txt
streamlit run app/streamlit_app.py
